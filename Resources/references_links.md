# External Technical References & OSINT Registries

This folder indexes the foundational open-source threat intelligence registries used to build the threat audit portfolio[cite: 1]:

1. **ICANN Global Domain WHOIS Lookup System**
   * Target Domain Profile: `mozilla.org` & `mozilla.com`.
   * Core Purpose: Ownership validation and top-level DNS registrar path analysis.

2. **Shodan Internet-Connected Device Database**
   * Endpoint Query: [Shodan Hostname Search](https://www.shodan.io)
   * Core Purpose: Network mapping and open proxy port analysis.

3. **VirusTotal Threat Intelligence Core Hub**
   * Assessment Profile: [VirusTotal Domain Safety Check](https://www.virustotal.com)
   * Core Purpose: Evaluating domain reputation health across global security scanners.

4. **Wayback Machine Public History Archive**
   * Snapshot Traversal Link: [Wayback Mozilla Timeline](https://web.archive.org/web/20150718000034/https://www.mozilla.org/en-US/)
   * Core Purpose: Historical web application data gathering and design footprint review.

5. **LinkedIn Corporate Sizing Insights Hub**
   * Business Target Directory: Mozilla Company Registry
   * Core Purpose: Employee workforce verification mapping (1,772 current registered profiles).
