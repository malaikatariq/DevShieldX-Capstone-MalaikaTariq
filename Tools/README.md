# OSINT Toolset & Asset Intelligence Directory

The following industry-standard tools and platform utilities were deployed to safely compile defensive telemetry on the target organization:

### 1. Shodan Asset Search Engine
* **Purpose:** Network exposure analysis, open listener identification, and global web infrastructure mapping.
* **Metrics Tracked:** 848 global targets, listening ports (443, 8443, 80), and server proxy software (Nginx, OpenResty, Apache).

### 2. VirusTotal Threat Intelligence
* **Purpose:** Multi-scanner domain evaluation, historical IP correlation, and reputation state tracking.
* **Metrics Tracked:** Verified completely clean reputation score across 92 primary security vendors.

### 3. Anymail Finder & LeadIQ
* **Purpose:** Social OSINT footprinting and workforce email structure extraction.
* **Metrics Tracked:** Correlated `FLast@mozilla.com` (89%) and `First@mozilla.org` (51%) formatting schemas.

### 4. Wayback Machine (Internet Archive)
* **Purpose:** Tracking structural code modifications, legacy file links, and ancestral data structures over a multi-year horizon.

### 5. Advanced Google Web Crawlers (Dorking)
* **Purpose:** File exposure and login interface indexing audits using specialized URL pattern filters.
